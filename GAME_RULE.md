# 게임 기획서: Fruit Catcher (과일 받기 게임)

## 1. 게임 개요
하늘에서 떨어지는 맛있는 과일을 바구니로 받아내는 게임입니다.
중간중간 떨어지는 폭탄을 피해야 하며, 시간이 지날수록 난이도가 상승합니다.

## 2. 조작 방법 (Pose)
플레이어는 웹캠 앞에서 몸을 움직여 바구니를 이동시킵니다.

- **Class 1: `Left` (왼쪽)**
    - 고개를 왼쪽으로 기울이거나, 왼쪽 손을 듭니다.
    - 바구니가 화면의 **왼쪽(Layer 1)**으로 이동합니다.
- **Class 2: `Center` (가운데)**
    - 정면을 바라보거나 차렷 자세.
    - 바구니가 화면의 **가운데(Layer 2)**로 이동합니다.
- **Class 3: `Right` (오른쪽)**
    - 고개를 오른쪽으로 기울이거나, 오른쪽 손을 듭니다.
    - 바구니가 화면의 **오른쪽(Layer 3)**으로 이동합니다.

> **참고**: 원활한 인식을 위해 3가지 구역(좌/중/우)으로 확실히 구분되는 동작을 학습시켜야 합니다.

## 3. 게임 규칙 (Rules)

### 3-1. 아이템 종류
- **사과 (🍎)**:
    - 기본 점수 아이템.
    - 획득 시 **+100점**.
- **황금 사과 (🌟)**:
    - 보너스 아이템 (가끔 등장).
    - 획득 시 **+300점**.
- **폭탄 (💣)**:
    - 방해 아이템.
    - 획득 시 **게임 오버** (또는 라이프 차감).

### 3-2. 플레이 방식
1. 게임이 시작되면 바구니가 중앙에 위치합니다.
2. 하늘(화면 위쪽)에서 랜덤한 위치(좌/중/우)에 아이템이 생성되어 떨어집니다.
3. 플레이어는 포즈를 취해 바구니를 해당 위치로 옮겨야 합니다.
4. 바구니와 과일이 겹치면 점수를 획득합니다.
5. 폭탄을 받으면 게임이 종료됩니다.

### 3-3. 레벨 디자인
- **Level 1**: 느린 속도로 사과만 떨어짐.
- **Level 2 (500점 이상)**: 폭탄이 조금씩 섞여서 떨어짐. 속도 약간 증가.
- **Level 3 (1500점 이상)**: 떨어지는 속도가 빨라짐. 황금 사과 등장.

## 4. UI 구성
- **상단**: 현재 점수(Score), 레벨(Level), 남은 시간(Time - 선택 사항)
- **중앙**: 게임 화면 (바구니와 떨어지는 아이템)
- **하단**: 인식된 내 포즈 상태 (DEBUG용)
